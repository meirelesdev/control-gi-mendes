# Control Gi Mendes

> Sistema de controle financeiro desenvolvido para gerenciamento de receitas e despesas, com interface web em JavaScript.

## Stack

- HTML5
- CSS3
- JavaScript

## Prerequisitos

- Nenhum prerequisito especifico

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/control-gi-mendes.git
cd control-gi-mendes
```
